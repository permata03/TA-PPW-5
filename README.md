Kalkulator Interaktif

Proyek ini merupakan aplikasi kalkulator sederhana berbasis web menggunakan HTML, CSS, dan JavaScript. Kalkulator ini dibuat untuk menjalankan perhitungan matematika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian.

Selain perhitungan biasa, kalkulator ini juga memiliki fitur memori. Tombol MC berfungsi untuk menghapus memori, MR untuk menampilkan nilai memori, M+ untuk menambahkan angka ke memori, dan M- untuk mengurangi angka dari memori. Fitur CE digunakan untuk menghapus satu karakter terakhir, sedangkan tombol C untuk menghapus seluruh input.

Setiap hasil perhitungan yang dilakukan akan otomatis tersimpan pada bagian history. Riwayat yang ditampilkan hanya sampai lima perhitungan terbaru agar tetap rapi dan tidak memanjang ke bawah.

Kalkulator ini juga mendukung penggunaan keyboard. Angka dan operator (+, -, *, /) dapat langsung diketik dari keyboard. Tombol Enter atau = digunakan untuk menghitung, dan Backspace untuk menghapus satu karakter.

Struktur program terdiri dari tiga file utama yaitu:

index.html untuk tampilan kalkulator di browser, 
style.css untuk mengatur desain tampilan, dan 
script.js untuk menangani proses perhitungan dan fungsi tombol.

Aplikasi bisa langsung dijalankan tanpa instalasi apa pun. Cukup buka file index.html melalui browser dan kalkulator siap digunakan.
